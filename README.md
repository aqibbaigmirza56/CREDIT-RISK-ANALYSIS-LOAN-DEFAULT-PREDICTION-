📌 Credit Risk Analysis & Loan Default Prediction
🔍 Project Overview

This project predicts whether a loan applicant is likely to default using Machine Learning techniques. It includes end-to-end steps from data analysis to model deployment using Streamlit.

🧠 Key Concepts Used

Exploratory Data Analysis (EDA)

Supervised Learning (Logistic Regression, KNN)

Unsupervised Learning (K-Means Clustering)

Model Evaluation (Accuracy, Precision, Recall)

Model Deployment (Streamlit)

🛠️ Technologies & Libraries

Python

Pandas, NumPy

Matplotlib, Seaborn

Scikit-learn

Streamlit

Joblib

📊 Dataset

Loan Default Dataset containing features such as:

Age

Income

Credit Score

Loan Amount

Employment & Financial Details

🚀 Model Workflow

Data Cleaning & Preprocessing

Exploratory Data Analysis

Feature Selection

Logistic Regression Model Training

Model Evaluation

Model Deployment using Streamlit

🌐 Deployment (Local)
pip install streamlit
streamlit run app.py

✅ Output

Predicts Low Risk (Loan Approved)

Predicts High Risk (Loan Rejected)
